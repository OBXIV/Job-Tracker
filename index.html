Connect The Dossier

This is a one-time setup. Connect a free Firebase database and your data will automatically sync across all your devices — no import/export needed.

1
Create a free Firebase project
Go to console.firebase.google.com → Click Add project → Name it (e.g. IT Pipeline) → Click Continue → Turn off Google Analytics → Click Create project
2
Create a Realtime Database
In the left sidebar → Build → Realtime Database → Create database → Choose any location → Select Start in test mode → Enable
3
Secure database rules
In Realtime Database → click the Rules tab → replace everything with this → click Publish:
{ "rules": { ".read": "auth != null", ".write": "auth != null" } }
4
Enable Google sign-in
In the left sidebar → Build → Authentication → Get started → Google → Enable it → choose a support email → Save.
5
Get your app config
Click the ⚙️ gear icon (top-left) → Project settings → Scroll to Your apps → Click </> Web → Register with any nickname → Skip Firebase Hosting → Copy the firebaseConfig object shown on screen
6
Paste your config below and connect
Paste the full firebaseConfig object (curly braces and all) into the box below.
Your firebaseConfig
🔒 Your config is stored only on this device. Your data lives in your Firebase project.
